# DailyCrumbs - Fresh Pastries Daily

A pastry-themed website connecting customers with local bakeries.

## Project Overview

DailyCrumbs is a web application designed to serve both customers looking for fresh pastries and bakeries wanting to reach more customers. The website features a warm, inviting design with pastry-themed colors and intuitive navigation.

## Features

### For Customers
- Browse featured pastries from local bakeries
- Easy ordering system
- User authentication (login/signup)
- Responsive design for all devices

### For Bakeries
- Platform to showcase products
- Reach local customers
- Manage orders and inventory

## Technologies Used

- **Frontend**: HTML5, Tailwind CSS, JavaScript
- **Design**: Warm pastry-themed color scheme (amber tones)
- **Icons**: SVG icons for social media and navigation
- **Images**: Placeholder images from Picsum for demonstration

## Design Principles

1. **Separation of Concerns**: Structure (HTML), Style (CSS), Behavior (JavaScript)
2. **Consistent UI**: Unified color scheme, typography, and spacing
3. **User Experience**: Clear calls-to-action, intuitive navigation
4. **Responsive Design**: Works seamlessly on desktop, tablet, and mobile
5. **Accessibility**: Semantic HTML, proper form labels, keyboard navigation

## Color Scheme

- **Primary**: Amber (#f59e0b) - Warm, inviting, pastry-themed
- **Secondary**: Amber-600 (#d97706) - Complementary warm tone
- **Background**: Amber-50 (#fef3c7) - Light, creamy background
- **Text**: Amber-900 (#78350f) - Dark, readable text
- **White**: Pure white for cards and forms
